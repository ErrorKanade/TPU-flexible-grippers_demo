## 1. 项目概述
本项目旨在开发一套基于 42 步进电机的柔性夹爪系统，核心目标是实现“快速接近、安全夹持”，即在提升效率的同时，通过力反馈实现“夹起鸡蛋而不破碎”的智能化控制并最终接入到jetson nano系统中。

## 2. 系统架构
本系统采用分层控制架构，确保执行效率与安全性：

 感知层 (Perception): 通过 FSR (薄膜压力传感器) 或力矩传感器获取实时物理量。
 执行层 (Execution): Arduino Nano/STM32 负责 PWM 脉冲生成与电机底层驱动。
 决策层 (Decision): Jetson Nano 通过串口通信进行闭环算法计算，下发速度与限位指令。


## 3. 硬件选型与连接
 电机驱动: 42 步进电机 + 驱动器 (如 TB6600/DM542)
 反馈传感器: FSR 压力传感器（推荐）或力矩传感器
 人机交互: 双轴摇杆 (VRx/VRy) 用于测试调试与参数整定
 核心控制: Arduino + Jetson Nano

## 4. 关键控制逻辑 (闭环策略)
为实现柔性夹持，采用“三阶段”运动控制策略：

1.  快速搜寻 (High-Speed Approach): 摇杆触发时，电机高速闭合，缩短空程时间。
2.  软接触 (Soft Touch): 监测压力阈值变化，检测到微小压力即刻减速。
3.  动态保压 (Adaptive Holding): 实时监控力反馈数据，达到预设压力区间即停止，并实时修正以补偿鸡蛋形变。

## 6. 调试心得与避坑
 防抖动: 摇杆务必设置“死区 (Deadzone)”，避免电机在静止时产生抖动。
 电磁干扰: 驱动器电源线与传感器信号线应分槽走线。
 响应速度: 调整 `AccelStepper` 的 `setMaxSpeed` 与 `setAcceleration` 以匹配细分参数。
